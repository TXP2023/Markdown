| 命令   | 功能描述                               |
|--------|----------------------------------------|
| `pwd`  | 显示当前工作目录的绝对路径             |
| `cd`   | 切换当前工作目录                       | 
| `ls`   | 列出目录内容（文件和子目录）           | 
| `echo` | 输出文本或变量内容到标准输出           | 
| `mkdir`| 创建新目录                             | 
| `time` | 测量命令执行时间实际输出三个时间值：real - 实际流逝时间user - CPU用户模式运行时间sys - CPU内核模式运行时间|
